####
Bash
####
.. highlight:: bash

There are various ways of transmitting telemetry data from the command line.

Depending on the desired protocol, please read about:

- How to do :ref:`data acquisition using MQTT <daq-mqtt>`.
- How to do :ref:`data acquisition using HTTP <daq-http>`.
- For simulating a simple periodic measurement, see
  :ref:`send a sawtooth signal <sawtooth-signal>` in order to
  transmit a dynamic, slowly oscillating sawtooth signal from
  the command line.
